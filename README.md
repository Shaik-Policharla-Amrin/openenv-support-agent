# OpenEnv Support Agent

Customer support simulation environment using OpenEnv.

## Endpoints

- `/reset` → get initial state  
- `/step` → take action  
- `/state` → current state  

## Description

This environment simulates real-world customer support tasks where an AI agent responds to user tickets and gets graded based on response quality.
